The page you're looking at

My largest progest so far is probably actually the stylesheet that governs the main page of this site (this page runs that stylesheet with a few quick additions). Its purpose is to display any page written with semantic HTML in a beautiful dark theme with well-structured content. It has everything from CSS Grid to pseudo-elements to media queries, and I believe it has unique rules for every single semantic tag (at least, all the ones in W3 schools' list). I'm pretty happy with it!

(Also, integrating all these programs with this page was a project in and of itself, which probably took more time than it took to write all of them. I improved them all a bunch during this process too.)
