A Caesar Cipher

This one is pretty self-explanatory. I made a Caesar Cipher class that lets you create an object with a given shift, and this object has methods that let you encrypt and decrypt text at will. Here in this box, you have an input field attached to an instance of this class which has the classic shift of "13". Try encrypting and decrypting any messages you want!

A Number Formatter

A neat little function that, relatively efficiently, takes a number, turns it into a string, and adds commas in the correct places such that it is transformed into a (you guessed it) formatted number (commas in-between every three characters). This thing even works with decimals.
